# Bloodpop
A blood bank management project i did to learn imgui basics

*WARNING!!* the codebase is a giant clusterf*ck. read at your own risk.

## Prerequisites
``sudo pacman -S sdl2``

## Installation
```
    $ git clone --depth=1 https://github.com/krolyxon/bloodpop --branch=master
    $ cd bloodpop
    $ make run
```
